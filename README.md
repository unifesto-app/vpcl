# VPCL - Vibe Coding Premier League

**India's First Franchise-Based Vibe Coding Competition**

## Overview

VPCL is a competitive coding league inspired by franchise sports leagues. Developers get drafted into teams, compete across a season, and the best builders become champions.

## How It Works

1. **Register** - Join the player pool
2. **Get Drafted** - Teams build their squad through the VPCL Auction
3. **Compete** - Face off against other teams in build challenges
4. **Climb the Table** - Win matches and qualify for the playoffs
5. **Become Champions** - Compete in the finals for the title

## Tech Stack

- React 19
- Next.js 16
- TypeScript
- Tailwind CSS
- Custom Agrandir fonts
